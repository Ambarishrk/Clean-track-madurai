# **App Name**: SyncSphere

## Core Features:

- User Authentication: Secure user registration and login using email/password and Google authentication, leveraging Firebase Authentication.
- Create Text Post: Allow authenticated users to create and publish new text-based posts to their profile and the global feed.
- Image Upload to Post: Enable users to upload images to accompany their posts, stored efficiently in Firebase Storage.
- Posts Feed Display: Render a dynamic feed of all user-generated posts, retrieved and displayed from Firestore.
- AI-powered Post Idea Generator Tool: Offer an AI-powered tool to suggest creative post ideas or content enhancements based on user input, assisting with content creation.
- User Profile Page: A dedicated page displaying the authenticated user's profile information and a collection of their published posts, all managed via Firestore.

## Style Guidelines:

- Primary color: A vibrant yet calming blue, signifying professionalism and approachability. Hex: #3373CC.
- Background color: A very light, desaturated blue-grey to provide a clean and subtle backdrop, ensuring readability. Hex: #EEF1F5.
- Accent color: A bright, clear aqua for highlighting interactive elements and calls to action, providing refreshing contrast. Hex: #2AB3DA.
- Headline and body font: 'Inter', a modern grotesque-style sans-serif, chosen for its neutral, objective, and highly legible appearance suitable for content-heavy applications.
- Utilize simple, clean line icons that convey meaning clearly without distracting from the content, aligning with a modern, minimalist aesthetic.
- Employ a clean, grid-based layout for displaying posts and content, ensuring visual organization and optimal use of space, with generous whitespace to enhance focus.
- Implement subtle, fluid animations for UI transitions and feedback on user interactions, creating a smooth and responsive experience without being intrusive.
